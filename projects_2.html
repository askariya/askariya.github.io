BASA DBMS

This is a DBMS that was implemented to deal with an imaginary airlineâ€™s basic transactions in a reliable and logical manner while also providing a user interface for the different types of users that need to access and alter information concerning the airline and its components. The DBMS is implemented as a website that provides a user interface for three different types of end users: Customers, Employees and Managers. The implementation uses a mix of SQL queries and associative arrays to retrieve and handle the database information and allow communication between the end user and the database. The database itself contains a relation for each type of end user and the website allows each end user to have different permissions and functionality.
